# multichat
Spike for acquaintance with some java technologies.

Chat which can use SOAP, JMX, RabbitMQ, REST for messages delivery.

---

Two types of messages: 
- text
- picture

---

Clients on:
- JavaEE (Wildfly) + JSF 
- Spring + GWT
 
---

Not a goal:
- fault tolerance
- visual design
- testing
